Principais formatações

Negrito / Destaque

Nesta frase, possuimos um termo em negrito usando a tag <b> (sem semântica).

Nesta frase, temos um termo em DESTAQUE usando a tag "<strong>" (semântica)


Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (sem semântica).

Nesta frase, temos um termo com ênfase usando a tag <en> (com semântica).


Texto Marcado

Podemos criar um texto marcado usando a tag <mark>


Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo (tag: <big> e <small>)


Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. (tag: <del>)


Texto Inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. (tag: <ins>)


Texto Sobrescrito

Para inserir coisas do tipo "x20+3". (tag: <sup>)


Texto Subscrito

Para inserir coisas do tipo "H2O". (tag: <sub>)
